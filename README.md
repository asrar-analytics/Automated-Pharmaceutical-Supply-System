# Automated Pharmaceutical Supply System 2025

![Dashboard]Dashboard.png

## Project Overview
This project is an interactive Power BI dashboard designed to support pharmaceutical inventory management and reduce drug shortages.

The dashboard helps monitor inventory levels, reorder points (ROP), and average daily demand to support better inventory decisions.

## Tools Used
- Microsoft Excel
- Power Query
- Microsoft Power BI

## Dashboard Features
- Current Stock by Medicine
- Reorder Point (ROP) by Medicine
- Average Daily Demand by Medicine
- Order Decision Overview
- Interactive Medicine Filter (Slicer)
